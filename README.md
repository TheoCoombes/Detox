# Detox
Userscripts that stop social media mobile sites from being so addictive, tested in Safari on iOS.

## Features

- **Instagram**: Removes ads, reels and the explore page. The page also slowly fades out to avoid excessive scrolling.
- **YouTube**: Completely removes YouTube shorts from home/navigation and redirects all shorts links to standard video links.
- **Reddit**: The page fades out to avoid excessive scrolling. Used in conjunction with [this Safari extension](https://apps.apple.com/us/app/sink-it-for-reddit/id6449873635).

## Installation

1. Firstly install [Userscripts](https://itunes.apple.com/us/app/userscripts/id1463298887) for Safari, or an equivalent monkeypatch extension.
2. Open the raw links to each `*.user.js` file in the repo in your browser.
3. Open the Userscripts extension popup and you will see an installation prompt.
4. Repeat 2-3 for all files.
